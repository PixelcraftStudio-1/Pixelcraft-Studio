Cómo desplegar en GitHub Pages

  1. Crear repositorio nuevo en GitHub (ej. pixelcraft-studio).
  2. Subir estos archivos a la rama main.
  3. Ir a Settings → Pages y seleccionar la rama main (root).
  4. Esperar unos minutos y abrir https://<tu-usuario>.github.io/pixelcraft-studio/.
